## 计算圆锥的体积
### Description
设 $\pi$ 的值为 3.14，输入圆锥的底面半径 $r$ 和高 $h$，求圆锥的体积 $v$。计算公式为 $v = \frac{1}{3}\pi r^2h$；设 $r, h$ 和 $v$ 的类型为 `double`。输出计算结果并保留小数点后 2 位，之后再输出 `\Everything is ok/`。记得最后要换行。
### Input
圆锥的半径 $r$ 和圆锥的高 $h$
### Output
圆锥的体积 $v$

## 求圆的直径、周长和面积
### Description
输入一个正整数 $n$ 作为圆的半径，求圆的直径、周长和面积，保留两位小数。
所有输入数据均满足 $0 < n < 10000$ 且 $n$ 为整数。
$\pi$ 取 3.14。
### Input
一个正整数 $n$
### Output
按格式输出直径（dia）、周长（circ）和面积（area），每行一个。末尾应当包含一个换行符（`\n`）。

## 最大公约数

### Description
计算两个整数的最大公约数 (Greatest Common Divisor)。如果两个整数都是正数，打印出它们的最大公约数。如果两个整数的其中任何一个为零或负数，打印 `invalid`。
### Input
两个整数 $a, b$
### Output
最大公约数或 `invalid`

## 构成三角形

### Description
编写一个程序，读取一个整数 $n$，然后读取接下来的 $n$ 组数据，每组数据包含三个在区间 $(0, 20]$ 里的实数。判断以这三个实数作为边长的三条边能否构成一个三角形。如果能，则输出 "True"；如果不能则输出 "False"。

构成三角形的充分条件：“任意两边之和大于第三边”。
### Input
第一行输入一个整数 $n$ ($1 \le n \le 10$)。
接下来 $n$ 行每行输入 3 个正数，每个数都在区间 $(0, 20]$ 内。
### Output
输出 $n$ 行，如果第 $i$ 组数据能够构成三角形，则输出 "True"，否则输出 "False"。输出的末尾应当包含一个换行符（`\n`）。

## 回文数

### Description
回文数是正着读和逆着读都相同的数。比如 7, 1221, 12321 都是回文数，而 1234 则不是回文数。现在给出一些数，请你判断它们是不是回文数。
输入是 $n$ 个需要你判断的整数 ($0 < n \le 100$)，每个数的范围是 $[0, 100000000]$。输入以 -1 结束。
### Input
若干个整数，以 -1 结束。
### Output
输出 $n$ 行，每一行输出一个 "Yes" 或 "No"，代表对应的数字是不是回文数。

## 选择性求和

### Description
用户输入一个小于 1000 的正整数 $n$ ($0 < n \le 1000$)，输出从 1 到 $n$ 里面所有能被 3 或 7 整除的数之和（结尾无换行）。
### Input
Sample input: 10
### Output
Sample output: 25
（解释：在 1 到 10 中，能被 3 整除的数为 3, 6, 9；能被 7 整除的数为 7；3+6+9+7=25，所以输出为 25。）
